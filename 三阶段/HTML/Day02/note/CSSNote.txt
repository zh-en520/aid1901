一、CSS基础

1.什么是CSS?

CSS(Cascading Style Sheet),层叠样式表

CSS的最新版本为CSS3(简称C3)

2.CSS有什么用?

CSS用于完成页面元素(如文本、图表、链接、表格等)的修饰(如字体、字号、颜色、边框、背景等);

3.CSS语法结构(内部样式)

<style type="text/css">

   选择器{
   		属性:值;
   		属性:值;   		
   		....
   }

</style>

二、选择器

1.通配选择器 -- *

2.类型或标记或元素选择器 -- 标记名称

3.类选择器 (A)定义类选择器,类名称必须以点(.)开头;
	    
	       (B)为使用类选择器的HTML标记添加class="类名称"属性

4.群组选择器 -- 选择器,选择器,....

5.后代选择器 -- 选择器 选择器 选择器 ...(要求选择器之间至少为父子关系)

三、BoxModel(盒子模型)

实际宽度 = 左侧外边距 + 左侧边框线 + 左侧内边距 + 宽度 + 右侧内边距 + 右侧边框线 + 右侧外边距


四、浮动

标准文档流:指行内元素将水平排列,块级元素将垂直排列;

float

描述:设置对象的浮动

语法:float:left|right;

说明:

A.对象浮动后将脱离标准文档流(也就是指在页面中不占用空间)

B.对象浮动后将尽量向左/向右移动,直到遇到父对象或上一个浮动对象的边界为止;

当某个元素的所有子元素都在浮动属性时,父元素高度将清除为零(也称为父元素的塌陷)

清理浮动

A.在父元素的未尾添加空的div元素(或者是其他块级元素);

B.定义CSS类(名称建议为clear或clearfix),具体如下:

  .clear{
  		clear:both;
  }

C.为刚刚新添加的div元素添加class="clear"属性
